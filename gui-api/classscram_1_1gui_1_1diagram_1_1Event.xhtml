Detailed Description

The base class for probabilistic events in a fault tree.

The base event item provides only the boxes containing the name and description of the event. A derived class must provide the symbolic representation of its kind.

The sizes are measured in units of character height and average width. This class provides the reference units for derived classes to use. All derived class shapes should stay within the allowed box limits to make the fault tree structure layered.
