Launches the processing of decomposition setups in preprocessing.

Processes common nodes in decomposition setups. This function only works with DecomposeCommonNodes() because it requires a specific setup of descendant marks and visit information of nodes. These setups are assumed to be provided by the DecomposeCommonNodes().
