Detailed Description

Manages XML element indentation upon output.

Precondition
Indenter instance is not shared between XML Streams.
Note
The indentation is bounded by kMaxIndent. If indentation exceeds kMaxIndent, only kMaxIndent indentation will be produced w/o error.
